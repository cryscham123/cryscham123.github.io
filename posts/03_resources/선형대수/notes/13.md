- 전치행렬의 행렬식은 원래 행렬의 행렬식과 같다.
- $(ABC)^T = C^T B^T A^T$
- $(A + B)^T = A^T + B^T$
- $(A^{-1})^T = (A^T)^{-1}$
- $C(A^T) = N(A)^⟂$
- $N(A^T) = C(A)^⟂$
- rank(A) + nullity(Aᵀ) = n

- $P=A(A^TA)^{−1}A^T$: projection matrix
- $Px$는 x를 A의 column space로 projection한 것이다.
- $A^TAx = A^Tb$: || Ax = b ||의 최소제곱해를 구하는 식
- 닮음 변환: $D=C^{-1}AC$ (C는 invertible matrix)
