## Machine Learning Process

1. Data Pre-Processing
    - import data
    - clean data
    - split data trainig and testing
    - feature scailing
        - normalization: $\frac{x - min(x)}{max(x) - min(x)}$
        - standardization: $\frac{x - μ}{σ}$
1. Modeling
    - build / train model
    - make predictions
1. Evaluation
    - calculate performance metrix
    - make a verdict
